# Support

Need help with a Malfence project? Here's how to reach us.

## 📖 Documentation
Start with the [docs site](https://github.com/malfencelab/docs) or the project's README.

## 💬 Community
- **Discussions** — Use the relevant repo's [Discussions](https://github.com/orgs/malfencelab/discussions) tab for questions and ideas.
- **Issues** — Use GitHub Issues for bug reports and feature requests.

## 🔐 Security Issues
Don't open public issues. See [SECURITY.md](SECURITY.md).

## 📧 Direct Contact
- General: `opensource@malfence.com`
- Security: `security@malfence.com`
- Code of Conduct: `conduct@malfence.com`

## ⏱️ Response Times
Maintainers are volunteers — please be patient. We aim to triage new issues within a week.
