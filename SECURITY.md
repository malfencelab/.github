# Security Policy

This repo follows Malfence's org-wide security policy.

→ **Full text:** [malfencelab/.github · SECURITY.md](https://github.com/malfencelab/.github/blob/main/SECURITY.md)

## Reporting Vulnerabilities

**Do not open public issues for security bugs.**

- 📧 Email: `security@malfence.com` (PGP key on [malfence.com/security](https://malfence.com/security))
- 🔒 Or use the **"Report a vulnerability"** button on this repo's Security tab.

We acknowledge reports within 48 hours and follow coordinated disclosure.
