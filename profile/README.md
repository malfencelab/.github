## 👋 About Us

Malfence is an open-source cybersecurity lab focused on **malware defense**, **binary analysis**, and **secure communications**. We build tools that make security accessible — free to use, free to study, free to improve.

Our mission is simple: **make modern threat defense available to everyone**, from individual developers to enterprises.

## 🚀 Featured Projects

| Project | Description |
| --- | --- |
| 🦠 [**malfence**](https://github.com/malfencelab/malfence) | A free and open-source antivirus solution, written in Rust. |
| 📬 [**maildfence**](https://github.com/malfencelab/maildfence) | A free, open-source mail gateway malware scanner. |
| 🔍 [**magika**](https://github.com/malfencelab/magika) | Fast and accurate AI-powered file content type detection. |
| 🧬 [**LIEF**](https://github.com/malfencelab/LIEF) | Library to instrument executable formats (PE, ELF, Mach-O). |
| ⚙️ [**DynamoRIO**](https://github.com/malfencelab/dynamorio) | Dynamic instrumentation tool platform. |
| 🧪 [**QBDI**](https://github.com/malfencelab/QBDI) | Dynamic Binary Instrumentation framework based on LLVM. |
